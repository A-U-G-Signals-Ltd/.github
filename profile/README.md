Website: https://augsignals.com/

## Overview

Founded in 1986, A.U.G. Signals Ltd. (AUG) specializes in:

- Signal processing
- Image and data processing
- Multi-sensor information fusion
- Intelligent transportation systems
- Environmental monitoring
- Remote sensing
- Safety, security, and defence technologies

AUG develops customized technologies and end-to-end solutions for government, academic, and industrial clients.

# Core Areas

## Remote Sensing Image Processing

AUG develops airborne and satellite-based hyperspectral, multispectral, and SAR processing technologies for:

- Mineral exploration
- Oil and gas mapping
- Precision agriculture
- Forest management
- Wildfire management
- Maritime traffic monitoring

### Capabilities

- Hyperspectral data analysis
- Multi-sensor image registration
- Image fusion
- Change detection
- Online distributed processing
